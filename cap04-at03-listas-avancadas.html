Seção 3 — Glossário Técnico

HTML
Linguagem de marcação usada em páginas web.
CSS
Linguagem de estilização visual.
JavaScript
Linguagem de programação da web.
Permite interatividade nas páginas.
DOM
Estrutura dos elementos HTML.
API
Interface de Programação
Comunicação entre sistemas.
HTTP
Protocolo de transferência web.
URL
Endereço Web
Localização de um recurso online.
DNS
Traduz domínios em IPs.
